Add Analysis Tools
The second step of the notebook generation process is to add computational tools to extract biological knowledge from the data.
The form below can be used to select pre-configured analysis tools to add to the notebook. A default selection of computational tools is provided. For more information on each tool, click on the icon on the right of each card.
Click on a tool to add or remove it from the notebook. Once the desired tools have been selected, click Continue to proceed.
Note: Users are welcome to submit their own analysis tools for integration. If you are interested in submitting your tool, please visit the Contribute page.
