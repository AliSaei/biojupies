Reuse Notebooks
Note: this page is still under development.
1. Download Docker
2. Open the Docker Console
3. Pull the lastest Docker container: docker pull maayanlab/biojupies-notebooks
4. Run the docker container. docker run -it -p 8888:8888 -e NOTEBOOK_UID=qzmeZ0p4s maayanlab/biojupies-notebooks:v0.1
Note: replace the NOTEBOOK_UID parameter with the UID of the notebook you wish to re-run.
5. Visit localhost:8888 on your browser
6. Run, edit and download your notebook.
